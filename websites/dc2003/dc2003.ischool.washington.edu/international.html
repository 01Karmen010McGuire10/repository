Preliminary Agenda:

Dublin Core in multiple languages is a crucial requirement for the DCMI metadata registry. Translation of Dublin Core into non-English languages is an important topic from the viewpoint of usage of the Dublin Core Metadata Elelement Set (DCMES) throughout the world. In this session, the translation tool (http://wip.dublincore.org/translate/index.html) will be demonstrated followed by a discussion of both how it can be used and of future requirements for the application.

  • "DCMI Registry: Multilingual functionality and the translation tool (tentative)"
    by Harry Wagner (OCLC, USA) and Mitsuharu Nagamori (Univ. of Tsukuba, Japan)
  • "Dublin Core in Multiple Languages -- Review of issues and current progress"
    by Thomas Baker (Fraunhofer-Gesellschaft, Germany)
