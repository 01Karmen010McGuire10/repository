Deposit

From DigiRepWiki

This wiki contains the following pages relating to deposit:

  • Deposit API - The Deposit API activity was convened by the Repositories Research Team in 2006 to explore options for developing a standard interface for depositing content into repositories.
  • SWORD (Simple Web service Offering Repository Deposit) - The SWORD project was funded in 2007 to take forward this unfunded activity and agree a standard mechanism for deposit.
