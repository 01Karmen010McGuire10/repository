Eprints DC XML/W3CXMLSchemas

From DigiRepWiki

[ Home | Functional Requirements | Application Model | Application Profile | Community Acceptance Plan | Mapping to Simple DC | XML Format

  • Schema: http://purl.org/eprint/epdcx/xsd/2006-11-16/epdcx.xsd
    • Target XML Namespace: http://purl.org/eprint/epdcx/2006-11-16/
    • declares the elements epdcx:descriptionSet, epdcx:description, epdcx:statement, epdcx:valueString, epdcx:XMLRepresentation, epdcx:binaryRepresentation
    • defines a set of named complex types that describe the default content models for these elements
    • declares the attributes epdcx:resourceURI, epdcx:valueURI, epdcx:vesURI, epdcx:sesURI, epdcx:representationURI, epdcx:resourceId, epdcx:valueRef
