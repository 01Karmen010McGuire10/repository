Introduction

This document defines the eprints-specific metadata properties and encoding schemes that have been created as part of the Scholarly Works Application Profile.

[edit]

Properties

The following properties have been created. Each term has been assigned a URI within the Eprint Terms (http://purl.org/eprint/terms/) namespace. Each term is usable as a property in metadata application profiles that conform to the DCMI Abstract Model. The description, domain and range indicate how the property should be used.

[edit]

Grant Number

Term URI 
http://purl.org/eprint/terms/grantNumber
Definition 
An alpha-numeric string identifying the funding grant under which the eprint was written.
Domain 
http://purl.org/eprint/entityType/ScholarlyWork
Range 
rdfs:Literal
Type of term 
Property
