Banners and Badges

The following banners and badges are provided in the JPEG file format for use by members of the public who wish to include them on a personal Web site.

No special permission is required from the Dublin Core Metadata Initiative to reproduce these images, however, banners must be linked directly to the Dublin Core Metadata Initiative Web site at http://purl.org/dc.

Please direct any technical questions about the use of these images to dc@oclc.org.
