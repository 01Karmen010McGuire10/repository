EIONET Meta Data Standard

  • Global Environmental Locator System Element Set (GELOS)

    • adopted from G7 working group on environmental metadata in late 1996
    • based on GILS
    • DC compliant
    • focus on both, discovery and description
    • contains qualifiers for contact information
    • however, datamodel and applications consider actors as individual resources

  • XML as exchange format
