Library uses of DC

  • Dublin Core serves internationally as an interchange format between various systems; supports reusability

  • Harvesting metadata from databases within and outside of library systems

  • Create library catalog records for Web resources within local ILS or within CORC

  • Expose MARC data to other communities through DC

  • Get data from other non-library creators
