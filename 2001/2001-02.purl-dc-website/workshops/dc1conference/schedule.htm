Small Working Groups

Preassigned groups will attempt to formulate a list of core elements that will comprise a useful record for resource discovery and retrieval of what is commonly understood as a document. The goal is to NOT to articulate a description record for arbitrary network objects, but it is hoped that the experience and judgement of the stakeholders will result in a list of data elements that will be naturally extensible to other object types.

Small Working Groups

Preassigned groups will refine the set of data elements from the previous day's work groups in light of the discussions of stakeholder's communities and special problems session (as well as hallway or bar conversations). The deliverable for this session is a matrix: data elements on one axis, communities on the other axis, with cells indicating the importance of each element to the various communities.
