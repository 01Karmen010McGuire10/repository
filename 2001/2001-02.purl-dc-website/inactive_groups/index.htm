Past Working Groups

As the Dublin Core Metadata Initiative matures, Working Groups are deactivated either because their work has been completed, the issues are no longer being addressed, or because the group has decided to change the focus of the issues. In any case, these deactiviated Working Groups remain part of the historical record of the Dublin Core Metadata Initiative.

Deactivated after the DC8 Conference (2000-10-06):

  • Coverage Working Group
  • Date Working Group
  • Datamodel Working Group
  • Format Working Group
  • Implementors Interest Group
  • Relation Working Group
  • One-to-One Working Group
  • Schema Working Group
  • Subject Description Working Group
  • Title Working Group

Deactivated after the DC6 Conference (1998-11-04):

  • Coverage Element Working Group
  • Relation_Type Working Group
  • Resource Type and Format Working Group
  • Rights Management Element Working Group
  • Subelement Working Group
