Open Issues:

  • Resource Type

    Dublin Core Resource Types: Open Issues - 1998-10-16

    Editor: Simon Cox, CSIRO Exploration & Mining, < Simon.Cox@dem.csiro.au >

    Please note that this is an informal list of current issues of which we are aware. The sequence and organisation are not necessarily significant.

    1. Cross-community study of existing lexicons and models of resource types
    2. are we happy with event?
    3. Subtypes
      1. text (huge hierarchy of document types)
      2. image (moving vs. still, symbolic graphics vs. representations)
      3. *.collection ?
    4. need to group type with accompanying format element for compound resources
    5. the 1:1 principle and compound/mixed resources

     

  • Format

    Dublin Core Format Element: Open Issues - 1998-10-16

    Editor: Simon Cox, CSIRO Exploration & Mining, < Simon.Cox@dem.csiro.au >

    Please note that this is an informal list of current issues of which we are aware. The sequence and organisation are not necessarily significant.

    1. types of format elements: format.size vs format.media
    2. structure of format.size:
      • different sizes - eg. mass vs linear, duration vs. bytes, pixels vs linear vs bytes
      • different aspects of a size - length vs breadth vs depth
    3. measurement units for format.size - is this a "scheme"?
    4. media types for non-electronic formats
    5. nested formats, eg. compressed, archives
    6. need to group type with accompanying format element for compound resources
    7. grouping multiple values of format together, eg size+media
    8. the 1:1 principle and compound/mixed resources
