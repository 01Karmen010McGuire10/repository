Working Group Charter:
The mission of the Working Group on Dublin Core in Multiple Languages is to coordinate the development of the Dublin Core as a multilingual metadata standard. As of June 1998, our focus is on creating a distributed registry of Dublin Core in multiple languages using the Resource Description Framework (RDF), an emerging standard syntax for expressing metadata schemas on the Web.
