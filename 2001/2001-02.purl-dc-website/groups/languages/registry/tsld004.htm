Registries as dictionaries

  • Metadata systems, like languages, need dictionaries for tracking usage and managing change

  • Like language dictionaries, registries can:

    • Prescribe good grammar and good usage guidelines
    • Describe how implementors are actually using metadata
    • Translate between natural languages
    • Define the "parts of speech" of metadata grammar -- building blocks of sentences
