Like languages, schemas evolve

  • Like words in languages, metadata terms may be coined, adopted, approved by official bodies, change meaning, or fall from use.

  • As in languages, need for simplicity is inevitably in tension with need for complexity

    • Dublin Core Element Set is almost always too simple to use "as is", so it is extended locally
    • If not managed, the proliferation of local extensions threatens interoperability in broader context
