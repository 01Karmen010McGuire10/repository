RDF as a publication format for schemas

  • How standards are are currently defined:

    • in HTML pages and paper documents
    • no explicit hyperlinks between related elements in different standards

  • RDF Schema format (based on XML)

    • URIs provide cross-references to related schemas and documentation
    • on Web, browse related namespaces (and profiles)
    • richer thesaurus relations will support crosswalks between elements that are not exactly equivalent
