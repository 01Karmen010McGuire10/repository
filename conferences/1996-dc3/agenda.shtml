Breakout Groups (See Breakout Document For Group Assignments)

As with other types of data, no monolithic element set will serve the needs of all users, but establishing a set of descriptors common to a wide range of image subtypes will promote interoperability and consistency of description across disciplines.

The focus of the first breakout disccussions is to identify a candidate set of elements to enhance resource discovery that is common to a broad variety of image types as the basis for an Image-Core element set.
