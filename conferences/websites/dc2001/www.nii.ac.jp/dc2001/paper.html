Preparing Your Paper

The Author Guidelines will provide complete information about paper format and style. Please follow these guidelines when preparing your paper.

If you plan to use LaTeX for compiling the manuscript, please take latex8.sty and adjust the margin settings referring to the sample LaTeX source file.

Clik on the file names below:

  • dc2001_guideline.txt: author guidelines (ASCII text)
  • dc2001_guideline.pdf: author guidelines (PDF) (created from dc2001_guideline.tex)
  • dc2001_guideline.tex: sample LaTeX source file
  • dc2001_guideline.bib: sample bibliography file
  • latex8.sty: style file to be used
  • latex8.bst: bibliography style file
See the page of 'Call for Papers and Participation'.
