Tidy

Tidy rules... plain and simple. Anyone interested in collaboratively editing and effectively managing HTML documents will benefit from the validation and formatting this program offers. I've found seemlessly integrating Tidy into Xemacs (which I used as my HTML editor) to be a very useful and time-saving addition to my personal "communication" toolkit.

So actually the function should be this:
 
; Function to run Tidy HTML parser on region 
; NOTE: this requires external Tidy program
(defun tidy-region ()
  "Run Tidy HTML parser on current region."
  (interactive)
  (let ((START (mark))
        (END (point))
        (COMMAND "tidy"))
        (save excursion
        (shell-command-on-region START END COMMAND t t  
shell-command-default-error-buffer))))
