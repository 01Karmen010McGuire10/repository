HTML 4.0

In HTML 4.0 The META element  the following attributes, the permitted values and their interpretation are profile dependent:

  • NAME: This attribute specifies a property name.
  • CONTENT: This attribute specifies a property's value.
  • SCHEME: This attribute names a scheme to be used to interpret the property's value.
  • HTTP-EQUIV: This attribute may be used in place of the name attribute. HTTP servers use this attribute to gather information for HTTP response message headers.
  • LANG: language information

The META element can be used to describe properties of a document (e.g., author, expiration date, a list of key words, etc.) and assign values to those properties. This specification does not define a normative set of properties.

The additional attributes SCHEME and LANG are useful for qualifying content. For example:

<META name="DC.creator" lang="fr" content="Arnaud Le Hors">
<META name="DC.subject" scheme = "DDC" content="541.34">
