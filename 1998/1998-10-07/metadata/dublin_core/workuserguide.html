Dublin Core Working Groups

Dublin Core User Guide Working Group

Moderator:  Bemal Rajapatirana (brajapat@nla.gov.au) and Diane Hillman (dih1@cornell.edu)

Charter:

This working group will be responsible for creating two user guides - one for Simple Dublin Core and another for Qualified Dublin Core.  The user guide will be published on the Web only, but will be freely available for use by users wishing to create guides for individual projects.

The target audience for both user guides will be individuals who have no training in cataloguing or indexing - a "Dublin Core for dummies" as it were.

Both guides will provide an introduction to Dublin Core including an overview of the basic principles governing the use of Dublin Core elements.  The guide will list the element names and their definitions, and outline how to create correct Dublin Core metadata for each of the Dublin Core elements with examples as required.   The guides will provide suggested sources for controlled vocabularies, as well as references to standard schemes applicable to particular elements where they may be useful.

The Qualified Dublin Core User Guide will extend theis work to include principles for the use of Dublin Core sub-elements, the list of approved sub-elements including names and definitions, and outline how to create correct metadata for each qualified Dublin Core element again with examples as required.

The guides will provide examples of complete Dublin Core records including examples of these records in the preferred syntax of HTML, either as stand-alone or embedded metadata.

The User Guide Working Group will consult other Dublin Core working groups to ensure that material in the guide is congruent with developments in the following areas:

Rights Management
Sub-Elements
DC.Date
Relation_Type
DC in Multiple Languages
Coverage
Format and Resource Types
