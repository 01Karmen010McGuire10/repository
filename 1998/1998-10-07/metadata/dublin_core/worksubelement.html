Dublin Core Working Groups

Sub-Elements Working Group

Moderator:  Paul Miller (collections@ads.ahds.ac.uk)

This working group exists to define the framework within which optional sub-elements may be added to the 15 elements of the Dublin Core. As well as defining suitable frameworks and procedures for addition, the group will construct an enumerated list of those sub-elements thought to be of generic value across the Dublin Core community.
