DCMI(Admin) Web Site CVS Access:

CVS Repository:

To access the CVS web site repository use the following CVSROOT:
   :pserver:guest@cvs.dublincore.org:/home/cvs

NOTE: The archive hostname has been changed. The current CVS archive for this project now resides on cvs.dublincore.org. Please forward any questions or comment to matola@oclc.org.

You may also use the login "guest" with password "guest" to gain read-only CVS access. If you have been given write access to the CVS repository, then replace "guest@cvs.dublincore.org" with your own login (e.g., "my_login@cvs.dublincore.org") and use your own tigris user password.

You need "password authetication" and "pserver port" (2401) set if you use WinCVS (see http://wincvs.org).

The modules to checkout are:

  • admin: The whole project, including the project web pages
  • admin/src: Source code for servlets and database access
  • admin/www: the project web site (www.dublincore.org).

The way most people contribute changes is to email a "update" to one of the "remote authors" (or "module owners"). These are authors who are responsible for certain sections of the web site. There is a list of authors here.
