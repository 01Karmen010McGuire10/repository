Admin Project Mailing Lists

To subscribe or unsubscribe, send an email message to the email address associated with each "subscribe" or "unsubscribe" line below. The subject line and message body are not used, so you may leave them blank.

Some of the email addresses on this page use the word "at" instead of an "@", this helps prevent spam.

You must subscribe before you can post to any of these mailing lists. This rule also helps prevent spam.

announce@admin: Announcements of new releases and other major project events.

Unsubscribe: announce-unsubscribe@admin.dublincore.org
