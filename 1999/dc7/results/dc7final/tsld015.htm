Data dictionaries (http://www.lib.washington.edu/msd/mig/datadicts/)

  • Group of library catalogers work with collaborators to set up data dictionaries.

  • Challenge: metadata consistency within and between image databases.

  • For each project data dictionaries document:

    • Database-specific field labels
    • Mapping of fields to DC elements
    • Data formatting instructions
    • Authority file (schema) recommended (if any)
