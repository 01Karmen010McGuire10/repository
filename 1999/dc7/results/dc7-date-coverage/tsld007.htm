Qualifiers for Coverage

  • Place Name

    • Qualified with an appropriate Value Qualifier

  • Box

    • A co–ordinate string, the structure of which is defined by the value qualifier
      • DC–Box–OSGB etc.
      • Recommended best practice to define East, West, North, South in that order, and to use Decimal Lat/Long (?), but it’s straightforward to do differently

  • Point

    • A single point in space, defined by co–ordinates
