Searching the Internet

  • A special program known as a “robot” (altenatively as a “scooter”, “drone”, “spider” or “web crawler”) explores the Internet and extract data about the resources it comes across;

  • the data extracted by the robot is stored in a data set on the search engine host computer (or computers) and is refined and structured to make it suitable for searches

  • the result is presented as an ordered set.
