<h2>Utilities</h2>
<a id="mkdoc" name="mkdoc"></a>
<p><a href="http://www.mkdoc.org/"><strong>MKDoc</strong></a></p>
<blockquote>
<p>MKDoc is a web site content management system that can produce HTML and RDF qualified Dublin Core metadata for every document. It also supports Dublin Core metadata via the generation of RSS 1.0 syndication feeds. MKDoc is free software and released under the General Public License.</p>
<p>All content in an MKDoc web site is managed via a web interface and it has been designed to make the creation of usable, accessible, semantically rich and valid web sites as easy as possible.</p>
<p>The following Dublin Core metadata can be set for every document:</p>
<ul>
<li>Lang</li>
<li>Title</li>
<li>Subject</li>
<li>Description</li>
<li>Publisher</li>
<li>Rights</li>
</ul>
<p>In addition MKDoc automatically generates the following Dublin Core metadata for documents:</p>
<ul>
<li>Identifier</li>
<li>Creator</li>
<li>Contributor</li>
<li>Date Created</li>
<li>Date Modified</li>
<li>References</li>
<li>Replaces</li>
</ul>
</blockquote>

<p><a href="http://www.splintered.co.uk/experiments/73/"><strong>Mozilla Firefox Dublin Core Viewer Extension</strong></a><br />This extension adds a button to the browser's status bar (and, optionally, the toolbar) to access an overview list of Dublin Core Metadata embedded in HTML/XHTML documents with META and LINK elements.</p>
<p><a href="http://www.ukoln.ac.uk/metadata/dcassist/"><strong>DC-assist</strong></a><br />A small, flexible help utility for metadata applications and is intended to complement the help pages embedded within existing software.</p>

<p><a href="http://www.illuminet.se/"><strong>IllumiNet Corpus</strong></a></p>
<blockquote>
<p>Corpus is a search engine written in Java that index content, context and metadata in documents on the network or on the local file system. It is used as a &quot;distributed relational database&quot; for document-oriented solutions with metadata as indexed keys.</p>
<p><strong>It also features:</strong></p>
<ul>
<li>Rich result overviews, structured by document context or by clustering</li>
<li>XML and RSS result output from built in HTTP server east to integrate with XSL transforms</li>
<li>Parser for multiple formats including HTML, XML-formats, PDF, DOC etc.</li>
<li>Auto generate metadata agent that uses LDAP and URL mappings</li>
<li>Direct indexing using WebDAV log or file system auditing events.</li>
<li>Web and command line administration</li>
<li>API's for new agents, spiders and parsers</li>
</ul>
