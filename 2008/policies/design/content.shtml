Content Area

The content area usually contains a main layout table and possibly smaller layout tables for menus.

One column

One column content areas do not include a layout table. Used for third and lower level pages.

Two columns

Consists of a one row, two column layout table. Used for second level pages only.

Element type Table
Summary Layout table: The first cell contains a navigation bar of links relavant to this page. The second cell contains the content.
Border 0
Height variable
Width 100%
Cellspacing 0
Cellpadding 5
Column one width 150 pixels
Column two width none specified
Column one background color #EEEEEE
Column two background color none specified
Column one horizontal alignment center
Column two horizontal alignment none specified

Three columns

Consists of a one row, three column layout table. Used for home page only.

Element type Table
Summary Layout table: The first cell contains a navigation bar of DCMI general information and ready references, the second contains news items, and the third a navigation bar of DCMI Working Groups.
Border 0
Height variable
Width 100%
Cellspacing none specified
Cellpadding none specified
Column one width 150 pixels
Column two width none specified
Column three width 150 pixels
Column one background color #EEEEEE
Column two background color none specified
Column three background color #EEEEEE
Column one horizontal alignment center
Column two horizontal alignment none specified
Column three horizontal alignment center
