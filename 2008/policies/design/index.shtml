Identity Elements

Overview

A brand's visual identity includes elements such as logo, color, typography and imagery. By consistently leveraging these elements, the Dublin Core Metadata Initiative presents a familiar and reliable brand to our users.

General Guidelines

Here is a helpful overview of the most important things you can do to help enhance and protect the DCMI brand identity:

  1. Be consistent. Inconsistent or unclear design can erode the brand.
  2. Use only approved DCMI logo artwork, and use it properly.
  3. Make it a priority to use the DCMI logo on all communications.
  4. Use Meta as a primary typeface for graphical text.
  5. Communicate positive messages that reflect our key message.
  6. When using imagery, be sensitive to our audiences and their cultures.
  7. Write in a style that is clear, friendly and knowledgeable.

Elements of the Design

The current Dublin Core Metadata Initiative website design consists of several common areas. Information on how to use the element is also included, using the following format:

Element Name The name of the element described.
ALT text The text used in the ALT tag (images only).
Image Name The naming/suggested naming convention for the image.
Image Location The location/suggested location on the server of the image.
Hyperlink The location the element links to.
Height The height of the element.
Width The width of the element.

Templates for the DCMI design have also been created to speed up your development process. They include all the common elements and appropriate server-side includes.

By using these templates, you can be assured that your authored pages will maintain the look and feel of the other pages on the DCMI website.
