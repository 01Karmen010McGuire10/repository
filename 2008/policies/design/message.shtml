Key Message

We have developed a key message to support and add relevance to our innovation position, and help give people around the world a consistent impression of our organization.

The key message is represented in italic text on the home page of the Dublin Core Metadata Initiative website:

Making it easier to find information using the Internet.

Focuses on:

Primary role of the Dublin Core Metadata Initiative.

Supporting points:

The DCMI is recognized worldwide as a leader in metadata standards.
