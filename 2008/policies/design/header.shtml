Header

The header appears at the top on every page of the Dublin Core Metadata Initiative website and gives the user a visual clue as to at where they are located, what sections are available for browsing, and a method of returning back to the home page.

The header should be used on ALL pages resident on dublincore.org. The header code should not be changed in any way. The header needs to appear at the top of the page and the code should be right after the opening BODY tag in each document.

Use the following header code for all pages (with the exception of the DCMI home page):

<!--#include virtual="/ssi/header.shtml"-->

Header Elements

DCMI logo

The DCMI logo is the main branding element on the DCMI website. It should always be referenced from the dublincore.org root level directory.

The DCMI logo should not be distorted in any way, and should always appear crisp. Bitmap versions of the DCMI logo must not be resized. Use the exact size of the supplied bitmap, or obtain new bitmaps from DCMI.

The logo always links back to the dublincore.org home page.

The DCMI logo used on the DCMI website is white on an orange background color field. No variations are permitted.

Color RGB Hex  
Orange 255, 102, 0 #FF6600  
Black 0, 0, 0 #000000  
White 255, 255, 255 #FFFFFF

Title

The website's title should appear in the bottom left corner of the header on all pages of the site. The slogan "Making it easier to find information." is only required to appear on the website's home page.

The title image always links back to the dublincore.org home page.

The font for the title on the Title image is 12 point Meta, with 6 point spacing. The text should be in upper and lowercase.

Bread Crumbs

Bread crumbs appear on every page of the DCMI website with the exception of the home page, and give the user a visual clue as to where the user is currently located. Bread crumbs are located on the bottom left of the header.

When a user is deep within the content, the bread crumb should reflect the section the user is browsing. For example, if the user is browsing a page within the DCMI Overview section, the bread crumb would reflect Home > About > Overview >.

The following bread crumb code is embedded within the header server-side include file (with the exception of the DCMI home page):

<!--#exec cgi="/cgi-bin/path.cgi"-->

Keyword Search

Keyword search appears on every page of the DCMI website including the home page, and gives the user a quick method of searching the website. Keyword search is located on the bottom right of the header.
