Footer

The footer appears at the bottom on every page of the Dublin Core Metadata Initiative website and gives the user information about the page's XHTML and CSS validation, a link to RDF metadata, copyright and privacy statements, and contact information.

The footer should be used on ALL pages resident on dublincore.org. The footer code should not be changed in any way. The footer needs to appear at the bottom of the page and the code should be right before the closing BODY and closing HTML tags in each document.

Use the following footer code for all pages:

<!--#include virtual="/ssi/footer.shtml"-->

Footer Elements

Horizontal rule

A horizontal rule separates the footer from the page content.

XHTML Validation

A badge that is linked to the W3C's Validation Service. Clicking on the link checks the HTML document for conformance to W3C HTML and XHTML recommendations and other HTML standards.

CSS Validation

A badge that is linked to the W3C's CSS Validation Service. Clicking on the link checks the HTML document for conformance to W3C for Cascading Style Sheets, level 2 standards.

Associated Metadata

A link to the RDF metadata associated with the currently displayed document.

Copyright and Privacy Statements

Information linking to DCMI Intellectual Property Notice and Legal Disclaimers.

Contact Information

Information linking to the DCMI Contact Information page.
