Frequently Asked Questions

Q. I want to use Microsoft FrontPage/Adobe PageMill/NetObjects Fusion on my site. Will the templates work with these programs?

A. FrontPage, Fusion and PageMill, like most WYSIWYG (What-You-See-Is-What-You-Get) HTML editors, will mangle HTML imported into it. We strongly suggest that you use editors which do not alter the template's code. Macromedia's Dreamweaver is such an editor (though it should be noted that even Dreamweaver can make unreadable code when used incorrectly).
