Naming Conventions

Directory, html, image, and media file names should ALWAYS be lowercase.

Spaces are not allowed. Avoid most punctuation, diacriticals or other accented characters. Names can include underscores and dashes if separate parts of a name need to be connected together.

The logic behind naming should be apparent.

There may be times when file names need to be named or numbered according to some other criteria, however, the logic behind the naming should be apparent.
