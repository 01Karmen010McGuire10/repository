WinCVS Preferences

1 After installing and launching the WinCVS application, select Admin -> Preferences in the main interface window.
2

The WinCVS Preferences dialog will appear.

From the General tab:

  • Enter the CVSROOT, substituting your user name for osbornel.
  • Select "passwd" file on the cvs server for the Authentication type.
3 Make any necessary changes to the Globals tab as shown below.
4 Make any necessary changes to the Ports tab as shown below.
5 Make any necessary changes to the Proxy tab as shown below.
6

From the WinCVS tab:

  • Select a default viewer used to open files. In reality, this is a default editor that you will be using to edit files. In this example, HomeSite has been chosen.
    Note: Microsoft Word is NOT recommended as an editor.
  • Make sure Use on double click has been checked.
  • Select a HOME folder on your hard drive (you may have to create a directory first). This is where WinCVS will store your password, as well as all of the files you will edit. In this example, we selected a directory named dublincore.org.

When finished making changes to the WinCVS Preferences, click the OK button.

You are now ready to Login.
