Adding files to WinCVS

1

To add a file, first place the file in the directory structure on your hard drive. This will be in the same directory (or a sub-directory) as the one you specified as your HOME folder in your WinCVS Preferences.

In the example below, a file named temp.shtml was added.
2

In WinCVS, navigate to the same directory in your HOME folder and select the added file.

Note: Notice that if a file has not been added to WinCVS, the file's icon will display as a white page with a question mark.
3

Select Modify -> Add selection.

Note: Add selection should be used for text files. If you are adding a PDF or image, select Add selection binary.
4 Once WinCVS has added the file, the file's icon will display as a red page with a letter 'A'.
You will now need to commit the file so that it may be published.
