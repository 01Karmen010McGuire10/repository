Logging into WinCVS

1 To login, select Admin -> Login.
2

The Password authentication dialog will appear.

Enter your password and click the OK button.
3

If your login failed, the result in the Output area will read:

*****CVS exited normally with code 1*****

If your login was successful, the result in the Output area will read:

*****CVS exited normally with code 0*****
You are now ready to checkout a module.
