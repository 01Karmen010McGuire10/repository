Header Elements

HTML Tag:
<H1></H1>
<H2></H2>
<H3></H3>
<H4></H4>
<H5></H5>
<H6></H6>
Guidelines:
Use header elements to convey document structure and use them according to specification. For example, use H2 to indicate a subsection of H1.
Warnings:
Do not use headers for font effects. Headers lower than H4 will usually display smaller in size than the default text.
Priority:
[Priority 2] A Web content developer should satisfy this checkpoint.
