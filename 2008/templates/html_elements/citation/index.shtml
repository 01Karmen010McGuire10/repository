Citation

HTML Tag:
<CITE></CITE>
Guidelines:
A CITE element is one that contains a citation or reference to some other source material. Browsers have free rein to determine how (or whether) to distinguish SITE element content from the rest of the BODY element. Both Navigator and Internet Explorer elect to italicize the text.

Make all visual distinctions as subtle as possible, but still clear and effective.
(Tufte, Visual Explanations)
