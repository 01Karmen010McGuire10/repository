Variable

HTML Tag:
<VAR></VAR>
Guidelines:
A VAR element is one that is used predominately to display one or more inline characters representing a computer program variable name. Browsers have free rein to determine how (or whether) to distinguish VAR element content from the rest of the BODY element. Both Navigator and Internet Explorer elect to render VAR element content in an italic font.
