Abbreviation

HTML Tag:
<ABBR TITLE=""></ABBR>
Guidelines:
Provides an encapsulation and enumeration mechanism for abbreviations that appear in the body text.
Warnings:
-
Priority:
[Priority 3] A Web content developer may address this checkpoint.

Real World Browser View Example:

OH

USA

Real World Source View Example:

<P><ABBR TITLE="Ohio">OH</ABBR></P>
<P><ABBR TITLE="United States of America">USA</ABBR></P>
