Strong Emphasis

HTML Tag:
<STRONG></STRONG>
Guidelines:
A STRONG element is one that contains text that indicates a stronger emphasis that the EM element. Whereas an EM element is typically rendered as italic text, a STRONG element is generally rendered as boldface text.
