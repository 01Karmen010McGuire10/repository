Defining Instance

HTML Tag:
<DFN></DFN>
Guidelines:
A DFN element signifies the first usage of a term in a document (its defining instance). A common technique in documents is to italicize an important vocabulary term the first time it is used in a document. This is generally the place in the document where the term is defined so that it may be used in subsequent sentences with its meaning understood. By default, Internet Explorer italicizes all text within a DFN element.
Warnings:
The DFN element is not supported in Navigator (up to version 4.x).
