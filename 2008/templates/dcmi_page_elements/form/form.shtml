Form

HTML Tag:
See examples below.
Guidelines:
Use OPTGROUP to group OPTION elements inside a SELECT. Group form controls with FIELDSET and LEGEND.
Warnings:
The FIELDSET element is not supported in Navigator (up to version 4.x).
The LEGEND element is not supported in Navigator (up to version 4.x).
The OPTGROUP element is not supported in either Navigator or Internet Explorer.
Priority:
[Priority 2] A Web content developer should satisfy this checkpoint.
