DataConversionDesign  
work to design expected output of a data conversion from MARC/MODS to RDF
Updated Feb 4, 2010

Design the expected output of a conversion from MARC/MODS to RDF using RDA and FRBR schemas for a set of sample records from the LoC dataset.

Verify the vocabulary (classes and properties) used in the output documents is defined in the RDA and FRBR schemas, document as issues any cases where this is not the case but cannot be easily resolved.

Write a test script to use the output documents as tests for the conversion stylesheet.

See MilestoneOne for the current scope.

Inspiration

Draw inspiration from the following...

  • DCMI/RDA cataloger scenarios
  • Semantic MARC
  • MARC to RDA mapping (RDA appendix D)
  • other RDA chapters and appendices...
